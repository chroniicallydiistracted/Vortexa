# /public Asset Kit (WestFam Weather)

This folder contains all browser-expected assets for icons, PWA, social previews, basic error pages, and SVG icon sets.

## Files
- PWA: manifest.webmanifest, android-chrome-192x192.png, android-chrome-512x512.png, maskable-icon-512.png, apple-touch-icon.png
- Icons: favicon.ico, icon.svg, wordmark.svg, safari-pinned-tab.svg
- Social: og-image.png (1200×630), twitter-card.png (1200×600)
- Error pages: 404.html, 403.html
- SVG sets: /icons/weather/*.svg, /icons/ui/*.svg
- Map assets: marker.svg, attribution.svg
- Snippet: HEAD-TAGS.html (paste into <head> of index.html)

## Colors
- Background: #0b0f14
- Theme: #0d1117
- Accent: #69b2ff
- Text on dark: #e8eef6
